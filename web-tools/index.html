Password Generator

16

Tips Keamanan Password:

  • Gunakan minimal 12 karakter untuk keamanan optimal
  • Kombinasikan huruf besar, kecil, angka, dan simbol
  • Jangan gunakan informasi pribadi (nama, tanggal lahir)
  • Gunakan password manager untuk menyimpan password
  • Aktifkan 2FA (Two-Factor Authentication) jika tersedia

Password Checker

Password Anda tidak disimpan atau dikirim ke server manapun

URL Safety Checker

Tanda-tanda Phishing:

  • URL yang mencurigakan atau typosquatting
  • Tidak menggunakan HTTPS (http:// alih-alih https://)
  • Domain yang mirip tapi tidak sama dengan situs resmi
  • Permintaan data pribadi atau password melalui email
  • Pesan mendesak atau ancaman
